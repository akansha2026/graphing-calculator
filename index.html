Interacting with the Application

Zooming

Activate Zoom: Hold down the Control key.

Zoom In/Out: Scroll up to zoom in and scroll down to zoom out. The graph will scale accordingly.

Panning

Click and Drag: Click on the canvas and hold the mouse button down.

Move: Drag the mouse to move the graph around.

Customizing Colors

Axes Color: Use the input field with the ID #axes to select a new color for the axes.

Graph Color: Use the input field with the ID #graph to select a new color for the graph.

Resetting the Graph

Double-Click: Double-click anywhere on the canvas to reset the graph to its original position and scale.
